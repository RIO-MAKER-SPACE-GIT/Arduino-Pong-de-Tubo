# Arduino Pong de tubo

Este projeto implementa uma versão do clássico jogo Pong utilizando dois microcontroladores Arduino:
* Arduino 1: Responsável apenas pela leitura dos sensores de proximidade, que funcionam como controles dos jogadores.
* Arduino 2: Executa toda a lógica do jogo e gera o sinal de vídeo composto para exibição em uma TV de tubo (CRT).

Funcionalidades
* Controle dos jogadores realizado por sensores de proximidade.
* Lógica do jogo e geração de vídeo totalmente implementadas em um segundo Arduino.
* Comunicação entre os Arduinos para garantir a interatividade do jogo.
